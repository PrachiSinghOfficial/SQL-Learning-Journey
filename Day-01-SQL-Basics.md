# 📖 Theory

## What is DBMS?

DBMS (Database Management System) is software that stores, manages, and organizes data efficiently.

Examples:
- MySQL
- Oracle
- PostgreSQL
- SQL Server

---

## What is SQL?

SQL (Structured Query Language) is used to communicate with databases.

It helps us:

- Create databases
- Create tables
- Insert data
- Retrieve data
- Update records
- Delete records

---

## Create a Database

```sql
CREATE DATABASE learning_sql;
```

---

## Use a Database

```sql
USE learning_sql;
```

---

## Basic SQL Query

```sql
SELECT DATABASE();
```

This query displays the currently selected database.
